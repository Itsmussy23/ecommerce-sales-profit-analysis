## Project Overview
This project analyzes an e-commerce dataset to understand sales and profit trends using Python.

## Objectives
- Analyze monthly sales trends
- Identify top-performing categories and sub-categories
- Compare sales and profit across customer segments
- Calculate sales-to-profit ratio for profitability analysis

## Tools & Libraries
- Python
- Pandas
- Plotly
- NumPy

## Key Analysis
- Converted date columns from object to datetime
- Extracted month, year, and weekday features
- Grouped data by category, sub-category, and segment
- Created interactive visualizations using Plotly

## Visualizations
- Line chart: Monthly Sales Trend
- Bar chart: Sales vs Profit by Segment
- Pie chart: Sales Distribution by Category

## How to Run
1. Clone the repository
2. Install required libraries
3. Run the Jupyter Notebook
